"MODTOHAFSD – A GUI based JAVA code for gravity modeling of strike limited sedimentary basins by means of growing bodies with exponential density contrast – depth variation".

The system requirements to  run the program are:-

Java Development kit(jdk)1.6 & above
RAM : 512 MB.
Disk space: 70 MB.


ZIP file details: -

The ZIP file consists of  two folders, a batch file, sample input and output data files, a readme text file,and an user manual.

Format of input file
-------------------------
Area Name or Profile name
Number of iterations
Number of Observations
Station spacing
Elevations
Observed anomalies
Minimum anomaly
Maximum anomaly
permissible error
Maximum permissible depth
Surface density contrast
Lambda
Offset values
Halfstrike lengths
Number of points selected for constructing analytically defined regional
x coordinates of selected points
y coordinates of selected points
Degree of polynomial
Estimated Polynomial coefficients 
Estimated regional background
